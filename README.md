# 📊 Attendance Management System

A full-stack web application to manage student attendance with role-based access for Faculty and Students.

---

## 🚀 Tech Stack

### Backend
- Java
- Spring Boot
- Spring Data JPA
- MySQL

### Frontend
- React.js
- Axios
- React Router
- Bootstrap

---

## ✨ Features

### 🔐 Authentication
- Role-based login (Student / Faculty)
- Secure access to dashboards

### 👨‍🏫 Faculty Features
- Add Students
- Add Subjects
- Mark Attendance
- View Attendance records

### 🎓 Student Features
- View subject-wise attendance
- View attendance percentage

---

## 🗂️ Project Structure
Attendance-Management-System
├── attendance-frontend
└── attendance-management-backend


---

## ⚙️ How to Run

### 🔹 Backend (Spring Boot)

1. Open project in Spring Tool Suite
2. Configure MySQL in `application.properties`
3. Run the application

Backend runs on:http://localhost:8085


---

### 🔹 Frontend (React)

```bash
cd attendance-frontend
npm install
npm start

Frontend runs on:http://localhost:3001
